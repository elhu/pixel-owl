# Pixel-Owl

## Synopsis

Provide pixelated owls to use as placeholders.

## Code Example

Simply use the "http://pixel-owl.com/pixel-owl" URL for your images, and you're set!

## Motivation

Owls are cool, I needed placeholders... meet Pixel-owl!
